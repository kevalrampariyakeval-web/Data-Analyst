# 📖 Project Story

Educational institutions collect large amounts of student data, including attendance, academic performance, classroom behavior, and subject-wise scores. However, without proper analysis, it becomes difficult to identify learning patterns and support student success.

To solve this challenge, I developed an interactive **Student Performance Dashboard** using **Power BI**. This dashboard provides a complete view of student performance by combining attendance, academic scores, behavior records, and individual student profiles into one interactive report.

The objective of this project is to convert raw educational data into clear visual insights that help teachers and school administrators monitor student progress and make informed decisions.

# 🎯 Project Objectives

This dashboard was created to answer important academic questions such as:

✅ How many students are enrolled?

✅ What is the overall attendance percentage?

✅ Which subjects have the highest and lowest average scores?

✅ Which students are top performers?

✅ Which students require academic support?

✅ How does student behavior affect overall performance?

✅ Which classes perform better academically?

✅ What are the attendance and behavior trends over time?
